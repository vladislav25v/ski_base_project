# Ski Base Site

Front-end for the ski school website. Includes public pages, news feed from Supabase, and admin login for content editing.

## Stack

- React + TypeScript + Vite
- Supabase Auth + Postgres

## Env

Create `ski-base-site/.env.local`:

```
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
VITE_ADMIN_UID=...
```

## Run

```
npm install
npm run dev
```
