# Ski Base Site

Волонтёрский сайт лыжной базы в городе Тында

Стек: фронтенд (React + Vite) и API (Express + Prisma + Postgres + S3).

## Локальный запуск

Инфраструктура через докер, endpoints в api/readme

1. Установить зависимости:

npm i
cd api && npm i

2. Запуск:

# API

cd api
npm run dev

# фронт (в другом терминале)

cd ..
npm run dev
